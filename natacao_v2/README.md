# Natação · Martes Freitas — Monitoramento

Sistema de monitoramento de desempenho de natação.

## Deploy no Railway

1. Faça upload deste projeto no GitHub
2. Conecte no Railway e crie novo projeto
3. Adicione um Volume em `/data` para persistir o banco de dados
4. Deploy automático

## Rodar localmente (Windows)

```
iniciar.bat
```

Acesse: http://localhost:5000
